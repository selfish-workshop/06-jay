# 와이어프레임 (텍스트 묘사)

각 화면을 ASCII로 대강 그려두고 핵심 영역만 설명한다. 디자이너가 Figma로 옮길 때 참고.

---

## S1. 모드 선택

```
┌─────────────────────────────────────────────┐
│                                             │
│            퇴근을 향하여                    │
│         오늘의 당신은 누구입니까?           │
│                                             │
│   ┌─────────────┐      ┌─────────────┐      │
│   │             │      │             │      │
│   │    🗡️        │      │    👑       │      │
│   │             │      │             │      │
│   │  용사 모드  │      │  보스 모드  │      │
│   │             │      │             │      │
│   │   Member    │      │   Leader    │      │
│   │             │      │             │      │
│   └─────────────┘      └─────────────┘      │
│                                             │
│       오늘의 퀘스트를    팀원의 전장을      │
│         수행하세요       확인하세요         │
│                                             │
└─────────────────────────────────────────────┘
```

- 배경: 다크 네이비 그라디언트 + 은은한 별/픽셀 노이즈
- 카드 호버 시: 살짝 확대 + 액센트 컬러 글로우
- 두 카드는 **반드시 시각적 무게 동일** (어느 모드가 중요하다는 편향 금지)

---

## S2. 캐릭터 생성 = 업무 입력

```
┌─────────────────────────────────────────────┐
│  🗡️ 용사 모드                    [모드 변경] │
│─────────────────────────────────────────────│
│                                             │
│      오늘의 용사를 소환하세요               │
│                                             │
│  ┌─────────────────────────────────────┐    │
│  │ 퀘스트명: [클라이언트 제안서 작성] │    │
│  │ 난이도:   ⭐⭐⭐⭐☆                   │    │
│  │ 타입:     [메인 ▾]                  │    │
│  │ 제한:     [D-2]                     │    │
│  │ 챌린지:   [대표님 리뷰 대기]        │    │
│  └─────────────────────────────────────┘    │
│                                             │
│              [ + 퀘스트 추가 ]              │
│                                             │
│                                             │
│              [  소환하기  ]                 │
└─────────────────────────────────────────────┘
```

- 입력 카드: 여러 개 추가 시 아래로 쌓임
- "소환하기" 버튼은 최소 1개 입력 전까지 비활성 (회색)
- 타입 드롭다운: 메인 / 서브 / 일일반복 / 보스전

## S3. 오프닝 시퀀스

```
┌─────────────────────────────────────────────┐
│                                      [Skip] │
│                                             │
│                                             │
│                                             │
│          202X년, 사무실이라는               │
│             던전에서                        │
│                                             │
│         용사들은 오늘도                     │
│         퀘스트를 받았다                     │
│                                             │
│       그들의 진짜 목표는 단 하나 —          │
│                                             │
│              퇴근                           │
│                                             │
│                                             │
└─────────────────────────────────────────────┘
```

- 배경: 완전 검정
- 텍스트: 흰색 + 픽셀 폰트 (Press Start 2P)
- 애니메이션: 아래→위 스크롤 (약 8초)
- Skip 버튼: 우상단, 반투명 배경, 항상 위 레이어

## S4. 퀘스트 메인 (용사모드 본 화면)

```
┌──────────────────────────────────────────────────┐
│  🗡️ 용사 모드                     [모드 변경] [⚙] │
│──────────────────────────────────────────────────│
│  오늘의 퀘스트   |  완료 1 | 진행 2 | 막힘 0     │
│──────────────────────────────────────────────────│
│                                                  │
│  ┌──────────────┐  ┌──────────────┐              │
│  │ [보스전]     │  │ [메인]       │              │
│  │ 제안서 작성  │  │ 광고 소재    │              │
│  │ ⭐⭐⭐⭐☆      │  │ ⭐⭐⭐☆☆      │              │
│  │ ████░░░░ 50% │  │ ████████ 100%│              │
│  │ D-2          │  │ D-Day  ✓클리어│              │
│  └──────────────┘  └──────────────┘              │
│                                                  │
│  ┌──────────────┐                                │
│  │ [서브]       │                                │
│  │ 경쟁사 리서치│                                │
│  │ ⭐⭐☆☆☆      │                                │
│  │ ██░░░░░░ 25% │                                │
│  │ D-3          │                                │
│  └──────────────┘                                │
│                                                  │
│              [ + 퀘스트 추가 ]                   │
└──────────────────────────────────────────────────┘
```

- 카드 클릭 → S4-1 모달
- 카드 타입 배지 컬러 구분: 보스전=레드, 메인=옐로, 서브=그레이, 일일=블루

## S4-1. 퀘스트 상세 모달

```
┌──────────────────────────────┐
│  제안서 작성        [✕]      │
│──────────────────────────────│
│  진행률:                     │
│  [0%] [25%] [50◉] [75%] [100%]│
│                              │
│  상태:                       │
│  [ ] 함정에 빠졌다 (막힘)    │
│                              │
│  보고 메모:                  │
│  ┌─────────────────────┐    │
│  │ 대표님 피드백 대기  │    │
│  └─────────────────────┘    │
│                              │
│           [ 저장 ]           │
└──────────────────────────────┘
```

## S5. 보스 대시보드

```
┌──────────────────────────────────────────────────┐
│  👑 보스 모드                      [모드 변경]   │
│──────────────────────────────────────────────────│
│  팀 전체 퀘스트 18  | 완료 4 | 진행 12 | 막힘 2🔴 │
│──────────────────────────────────────────────────│
│                                                  │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐          │
│  │ 김대리   │ │ 이과장🔴 │ │ 박주임   │          │
│  │ 제안서   │ │ 기획안   │ │ 리서치   │          │
│  │ ████░ 50%│ │ ██░░ 25% │ │ ███░ 75% │          │
│  │          │ │ [막힘]   │ │          │          │
│  └──────────┘ └──────────┘ └──────────┘          │
│                                                  │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐          │
│  │ 최사원   │ │ 정주임🔴 │ │ 한대리   │          │
│  │ 카피 작성│ │ 영상 편집│ │ 분석     │          │
│  │ █████100%│ │ █░░░ 10% │ │ ███░ 60% │          │
│  │ ✓클리어  │ │ [막힘]   │ │          │          │
│  └──────────┘ └──────────┘ └──────────┘          │
└──────────────────────────────────────────────────┘
```

- 막힘 뱃지: 빨간 점 + 카드 테두리 빨강
- 카드 클릭 → S5-1 모달 (해당 팀원의 전체 퀘스트 리스트)
- 팀원 순서: 막힘 → 진행중 → 완료 (중요도 순 정렬)

---

## S5-1. 팀원 상세 모달

```
┌──────────────────────────────────┐
│  이과장의 전장              [✕]  │
│──────────────────────────────────│
│  [보스전] 기획안       25% 막힘🔴│
│   메모: 대표님 피드백 대기       │
│  [메인]  회의록        완료 ✓   │
│  [서브]  자료 정리     50%       │
└──────────────────────────────────┘
```

- 보스는 **읽기 전용**. 팀원 데이터 수정 불가 (데모 안전)
