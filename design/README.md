## 핵심 결정

- **톤**: 픽셀 RPG × 모던 웹 UX 하이브리드 (완전 픽셀풍 X)
- **모드 구분**: 용사=옐로 액센트 / 보스=퍼플 액센트
- **주요 폰트**: `Press Start 2P` (강조) + `Pretendard` (본문)
- **반응형**: 1440×900 우선, 최소 1024px, 모바일 제외
- **Skip**: ESC·Space 키로도 가능 (발표 현장 편의)
