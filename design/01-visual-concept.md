# 비주얼 컨셉 — 퇴근을 향하여

## 핵심 방향

**"픽셀 RPG × 모던 웹 UX"** 하이브리드.
완전 픽셀풍이면 사용성 떨어지고, 완전 모던이면 컨셉이 약함.
→ **레이아웃은 모던, 디테일·장식은 픽셀풍**으로 간다.

## 톤앤매너

- **친근함 + 유머** — 직장인 공감 중심
- **경쾌함** — 무거운 업무 앱 느낌 배제
- **깔끔함** — 보스모드는 정보 밀도 높지만 답답하지 않게
- **과장 자제** — 도트 아이콘·네온은 포인트로만

## 컬러 팔레트

| 역할 | 색상 | HEX | 용도 |
|---|---|---|---|
| Primary | 다크 네이비 | `#1A1B3A` | 배경, 헤더 |
| Accent | 퀘스트 옐로 | `#F5C518` | 퀘스트 배지, 강조 버튼 |
| Success | 클리어 그린 | `#4ADE80` | 완료 상태 |
| Warning | 함정 레드 | `#EF4444` | 막힘, 위험 |
| Neutral | 회색톤 | `#E5E7EB` / `#6B7280` | 텍스트, 경계 |
| Boss Mode | 로열 퍼플 | `#8B5CF6` | 보스모드 헤더·액센트 |

**모드별 액센트 차별화**:
- 용사모드 = 옐로 액센트
- 보스모드 = 퍼플 액센트
→ 첫인상에서 두 모드 구분이 즉시 보임

## 타이포그래피

- **타이틀·강조**: `Press Start 2P` (Google Fonts) — 픽셀풍 게임 폰트
- **본문·입력**: `Pretendard` 또는 `Noto Sans KR` — 한글 가독성
- **숫자·수치**: `JetBrains Mono` — 진행률, 카운터 등

> 픽셀 폰트는 장식용. 본문에 쓰면 눈 아픔.

## 아이콘 스타일

- 16×16 픽셀 아트 아이콘 (퀘스트 타입, 상태)
- Lucide/Heroicons 등 모던 SVG는 UI 컨트롤에만 (닫기, 메뉴 등)
