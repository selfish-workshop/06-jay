# 퇴근을 향하여

업무를 RPG 퀘스트처럼 관리하고, 팀장은 보스 시점에서 팀원의 진행 상황을 확인하는 게임형 업무관리 웹서비스 데모.

## 프로젝트 정보
- **유형**: 웹서비스 데모 (업무관리 / UX 실험)
- **타겟 사용자**: 직원(용사모드) + 팀장·대표(보스모드)
- **출시 목표**: 워크숍 발표 시연용 프로토타입

## 핵심 기능
- 모드 선택 (용사모드 / 보스모드)
- 캐릭터 생성 단계로 위장한 오늘의 업무 입력
- 영화 크레디트 스타일 오프닝 시퀀스 (Skip 가능)
- 퀘스트 화면에서 업무 진행률·챌린지·난이도·일정 관리
- 보스모드 대시보드에서 팀원별 퀘스트 현황 한눈에 보기

## 폴더 구조 (5개 팀)
- `planning/` — 기획 팀: 무엇을 만들 것인가
- `design/` — 디자인 팀: 어떻게 보여줄 것인가
- `cx/` — CX 팀: 누가 어떻게 사용할 것인가
- `marketing/` — 마케팅 팀: 어떻게 알릴 것인가
- `strategy/` — 전략 팀: 어떻게 지속 가능하게 만들 것인가
