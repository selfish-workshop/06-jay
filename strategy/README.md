## 핵심 결정

- **수익 모델**: Freemium → Team SaaS → Enterprise 3단계
- **Phase 2 팀 플랜**: 월 ₩9,900~29,900 (경쟁 툴 절반 이하)
- **1순위 파트너**: 업무 메신저(Slack, Teams) 통합
- **의사결정 게이트**: M3(컨셉 검증) / M6(수익화) / M12(지속성)
- **장기 방향**: 실적관리 연동으로 HR Tech 확장

## 원칙

1. **검증 우선**: 수익화 전 컨셉 검증
2. **작게 시작**: 1인 팀으로 Phase 2까지 가능한 설계
3. **단계 게이트**: 각 시점마다 계속/중단/피벗 판단
4. **사회적 가치**: 보고 문화 개선이라는 비수익 가치 유지
