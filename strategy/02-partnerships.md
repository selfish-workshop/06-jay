# 잠재 파트너십

## 전략적 파트너 카테고리

### 1. 업무 메신저 통합
**대상**: Slack, Microsoft Teams, Discord, 카카오워크
**가치**:
- 막힘 상태 변경 시 해당 리더에게 메신저 알림
- 퀘스트 완료 시 팀 채널에 축하 메시지
- 슬래시 커맨드로 빠른 퀘스트 입력 (`/quest 제안서작성`)

**우선순위**: ⭐⭐⭐ (1순위)
**이유**: 기존 업무 흐름에 가장 자연스럽게 붙음

### 2. 캘린더·업무 툴 연동
**대상**: Google Calendar, Notion, Linear, Jira, Asana
**가치**:
- 캘린더 일정 → 퀘스트 자동 변환
- Notion 데이터베이스 동기화
- Jira 티켓 → 퀘스트 매핑

**우선순위**: ⭐⭐ (2순위)
**이유**: 사용자가 이미 쓰는 도구와의 호환성

### 3. HR·조직문화 플랫폼
**대상**: 리멤버, 잡플래닛, 사람인, 원티드
**가치**:
- "좋은 보고 문화를 가진 회사" 배지
- 조직문화 개선 사례 콘텐츠
- 채용 브랜딩 연계

**우선순위**: ⭐ (3순위, 장기)
**이유**: 수익 직결보다 브랜드 인지도

### 4. 기업 교육·컨설팅
**대상**: 퍼블리, 패스트캠퍼스, 인프런
**가치**:
- "업무 보고 개선" 주제 워크숍
- 조직문화 컨설팅 패키지
- B2B 영업 입구

**우선순위**: ⭐⭐ (2순위)
**이유**: 엔터프라이즈 영업 전환 경로

## 초기 접근 전략

### 단계 1: 개인 네트워크 활용 (0~3개월)
- 워크숍 참여자 및 주변 스타트업 대표에게 체험 요청
- 피드백 기반 개선 + 사례 확보
- 목표: 정성적 피드백 20건

### 단계 2: 개발자 관계 (3~6개월)
- Slack API, MS Graph API 연동 개발
- 각 플랫폼 파트너 프로그램 등록
- 목표: 1개 플랫폼 공식 연동

### 단계 3: 콘텐츠 파트너십 (6~12개월)
- 리멤버·퍼블리에 "업무 보고 문화" 기고
- 조직문화 컨퍼런스 발표
- 목표: 미디어 노출 3건

### 단계 4: B2B 영업 (12개월+)
- 엔터프라이즈 파일럿 3곳
- SSO·보안 인증
- 목표: 유료 기업 고객 5곳

---

## 파트너십 리스크

- **기능 종속**: 특정 플랫폼 API 의존도가 너무 높으면 위험
- **브랜드 훼손**: HR 플랫폼과 엮이면서 "감시 툴" 이미지 오염 가능성
- **영업 비용**: 엔터프라이즈 딜은 영업 인력 필요 → 1인 팀으로는 한계
