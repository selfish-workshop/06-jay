- **1순위 타겟**: 5~15인 스타트업 팀
- **핵심 채널**: 트위터/X + 디스카이엇 (제품 만드는 커뮤니티 중심)
- **금기 표현**: "업무 관리" "생산성" "감시"
- **포지셔닝**: 업무 관리 툴이 아닌 **업무 보고 부담 제거 툴**
