# 채널별 상세 전략

## 1. 트위터/X

**포맷**: 스레드 + GIF/영상
**목표 독자**: 스타트업 종사자, PM/디자이너/개발자

**포스팅 예시**:
```
스레드 1/5
업무 보고하기 싫어서 만든 서비스 🧵

제 팀원들은 아침마다 슬랙에 "오늘 할 일" 적어야 했어요.
근데 그거 쓰는 데 10분씩 걸렸거든요.

그래서 생각했어요 — "게임처럼 입력하면 덜 피곤하지 않을까?"

2/5 [GIF: 캐릭터 생성 화면]
캐릭터 만드는 화면처럼 생긴 업무 입력 폼이에요.
"오늘의 용사를 소환하세요"라고 하면
신기하게 업무 입력이 덜 지겹더라고요.

3/5 [GIF: 퀘스트 화면]
입력한 업무는 퀘스트 카드로 나와요.
난이도, 진행률, 막힘 여부...
실제 업무 정보를 게임 요소로 재해석했어요.

4/5 [GIF: 보스 대시보드]
근데 진짜 핵심은 여기예요.
팀장 모드로 바꾸면 팀원들 상황이 한눈에 보여요.
"지금 뭐 해?" 안 물어봐도 돼요.

5/5
워크숍 프로젝트로 시작했는데
실제로 팀에서 써보니 보고 시간이 반으로 줄었어요.

링크: [URL]
```

**KPI**: RT 10+, 좋아요 30+, 링크 클릭 50+

## 2. 인스타그램 릴스

**포맷**: 30초 세로 영상, 자막 필수
**목표 독자**: 20~30대 직장인

**영상 구성 (30초)**:
- 0~3초: 훅 — "보고서 쓰기 싫어하는 사람 주목"
- 3~10초: 문제 — 직장인의 아침 (슬랙에 보고 적는 장면)
- 10~20초: 솔루션 — 서비스 시연 (모드선택 → 업무입력 → 퀘스트)
- 20~27초: 보스모드 전환 순간 (반전 포인트)
- 27~30초: CTA — "링크는 프로필에"

**해시태그**: #직장인 #업무 #퇴근 #스타트업 #업무관리 #워크플로우 #오늘의업무 #막내생활

**KPI**: 조회 1000+, 저장 20+, 공유 10+

## 3. 링크드인

**포맷**: 긴 텍스트 + 스크린샷 3~5장
**목표 독자**: 리더·관리자·HR

**포스팅 톤**: 차분하고 신중. 유머 자제.

**포스팅 예시 구조**:
1. 문제 제기: "팀원 업무 파악, 어떻게 하고 계세요?"
2. 리더의 딜레마: 물어보기 미안, 모르면 불안
3. 우리 접근: 팀원 주도 입력 + 리더 읽기 전용 구조
4. 스크린샷 3장: 보스 대시보드 중심
5. 회고: "팀원들 반응이 좋았던 이유는..."
6. CTA: "관심 있으신 분은 댓글"

**KPI**: 반응 30+, 댓글 10+, 공유 5+

## 4. 디스카이엇 (Disquiet)

**포맷**: 메이커 로그
**목표 독자**: 제품 만드는 사람들

**주간 로그 템플릿**:
```
[이번 주 업데이트]
- 기능 추가: ___
- 배운 것: ___
- 다음 주 할 것: ___

[이번 주 숫자]
- 방문자: N명
- 가입: N명
- 피드백: N건

[피드백 요청]
- 여러분은 ___ 어떻게 생각하세요?
```

**KPI**: 댓글 5+, 반응 10+, DM 2+

## 5. 유튜브 쇼츠

**포맷**: 60초 시연 영상
**목표 독자**: 유튜브 생산성 콘텐츠 시청자

**구성**:
- 0~5초: 훅 — 썸네일과 첫 문장
- 5~15초: 문제 공감
- 15~45초: 풀 데모 (모드선택 → 입력 → 오프닝 → 퀘스트 → 보스모드)
- 45~55초: 강점 정리
- 55~60초: 링크 안내

**KPI**: 조회 500+, 구독 10+

---

## 공통 브랜드 톤

- **친근하지만 진지**: 장난기 있게 시작, 문제는 진지하게
- **과장 금지**: "혁신" "세상을 바꾸는" 같은 표현 X
- **공감 우선**: "여러분도 그렇죠?" 톤
- **데이터 기반**: 가능하면 숫자 포함 ("보고 시간 50% 감소" 등)
