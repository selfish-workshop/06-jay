# 온보딩 시나리오

첫 방문자가 "이게 뭐지?"에서 "오, 쓸만한데?"로 가기까지의 경험 설계.

---

## 설계 원칙

1. **튜토리얼 없음** — 게임처럼 경험하면서 이해
2. **첫 2분이 전부** — 2분 안에 가치 체감 못 하면 이탈
3. **텍스트 최소** — 설명은 짧게, 행동으로 학습
4. **되돌리기 쉬움** — 언제든 모드 변경으로 복귀

---

## 첫 방문 (0~60초)

### 0~5초: 모드 선택 진입
- 화면: 모드 선택 (S1)
- 사용자 생각: "이게 뭐지? 두 개 중에 뭐 고르지?"
- 장치:
  - 카드 아래 **한 줄 설명** ("오늘의 퀘스트를 수행하세요" / "팀원의 전장을 확인하세요")
  - 첫 방문자는 **용사모드 카드에 살짝 pulse 애니메이션** (기본 경로 유도)

### 5~15초: 용사모드 선택
- 화면: 캐릭터 생성 = 업무 입력 (S2)
- 사용자 생각: "캐릭터 만들라고? 어... 이거 업무 입력이네?"
- 장치:
  - 상단 카피 "오늘의 용사를 소환하세요"
  - 입력 필드 placeholder에 **실제 업무 예시** ("예: 제안서 작성")
  - → 사용자가 "아, 업무를 넣는 거구나" 자동으로 이해

### 15~45초: 퀘스트 입력
- 사용자 행동: 퀘스트 1~3개 입력
- 마찰 포인트:
  - 필드가 많으면 이탈 → **필수 3개만** (퀘스트명/난이도/타입)
  - 나머지는 선택
- 장치:
  - 첫 퀘스트 입력 완료 시 "소환하기" 버튼이 살짝 **펄스** (진행 유도)
  - "+ 퀘스트 추가"는 1개 입력 후에야 등장

### 45~55초: 오프닝 시퀀스
- 화면: S3
- 사용자 생각: "오, 영화처럼 나오네"
- 장치:
  - 첫 방문자는 **Skip 버튼이 5초 뒤에 등장** (한 번은 끝까지 보게)
  - 스킵 버튼 등장 시 살짝 페이드인

### 55~60초: 퀘스트 메인 첫 진입
- 화면: S4
- 사용자 생각: "아, 내가 입력한 게 카드로 나왔네"
- 장치:
  - 카드들이 순차 페이드인 (스태거)
  - 첫 방문자는 첫 카드에 **"클릭해보세요" 툴팁** 1회 노출

---

## 첫 성취 경험 (60~120초)

### 첫 진행률 업데이트
- 사용자 행동: 카드 클릭 → 진행률 25% 또는 50%로 조정 → 저장
- 결과:
  - 카드 진행률 바 부드럽게 애니메이션
  - 상단 카운터 업데이트 ("진행중 1")
- 사용자 감정: "어? 반응하네"

### 모드 전환 경험
- **강력한 첫 성취**: "모드 변경" 버튼 클릭 → 보스모드 → 본인 카드에 방금 입력한 데이터 반영된 것 확인
- 장치:
  - 보스모드 첫 진입 시 **본인 카드가 제일 위에 위치**
  - 카드 테두리 옐로 하이라이트 (본인임을 강조)
- 사용자 감정: "와, 실제로 연동되네"

→ **이 순간이 가치 체감의 정점**. 여기까지 오면 "이거 쓸만하다" 인지.

---

## 재방문 온보딩 (2일차 이후)

### 변화 포인트
- 오프닝 시퀀스: 자동 Skip 또는 전체 생략
- 캐릭터 생성: 어제 퀘스트 목록을 "어제의 퀘스트 다시 불러오기" 버튼으로 제안 (선택)
- 초기 로딩: 바로 S4 퀘스트 메인으로 진입 옵션 (설정에서 on/off)

### 유지 장치
- 일일 클리어 카운트 (단순 숫자만)
- "오늘의 퀘스트 완료 수" 상단 배지

---

## 이탈 포인트 대응

| 단계 | 이탈 이유 | 대응 |
|---|---|---|
| 모드 선택 | "뭐 고를지 모르겠다" | 한 줄 설명 + 용사 카드 pulse |
| 업무 입력 | "필드 너무 많다" | 필수 3개 + 선택 2개로 제한 |
| 오프닝 | "길다" | 첫 방문만 끝까지, 이후 Skip 기본 |
| 첫 카드 클릭 | "뭘 해야 할지 모름" | 툴팁 1회 노출 |
| 모드 전환 | "보스모드 왜 있어?" | 본인 카드 하이라이트로 연동 체감 |

---

## 발표 데모에서의 온보딩 시연

**시연자는 "처음 쓰는 사람"처럼 행동**:
1. 모드 선택에서 살짝 망설이는 척 → "아, 용사모드" 선택
2. 캐릭터 생성 화면에서 "어? 캐릭터 아니네?" 리액션
3. 업무 입력 → "소환하기"
4. 오프닝에서 "이거 재밌네" 리액션 후 Skip
5. 퀘스트 카드 진행률 조정 → "이렇게 되네?"
6. 모드 변경 → 보스모드 → "어, 내가 방금 한 게 여기 있네!" (클라이맥스)

**이 스토리가 5분 안에 끝나야 합니다.**
