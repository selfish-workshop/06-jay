# 고객 여정맵

용사(김대리) / 보스(박팀장) 두 축으로 **인지 → 진입 → 사용 → 재방문** 4단계 매핑.

---

## 용사 여정 — 김대리

| 단계 | 액션 | 감정 | 터치포인트 | 개선 기회 |
|---|---|---|---|---|
| **인지** | 팀 슬랙에서 링크 공유받음 | 😐 또 업무 툴..? | Slack 공유 | "게임" 키워드로 호기심 유발 |
| **진입 (첫 방문)** | URL 클릭 → 모드 선택 화면 | 🤔 이게 뭐지? | S1 모드 선택 | 용사/보스 카드 시각 임팩트 |
| **첫 사용** | 용사모드 → 캐릭터 생성 | 😊 어? 업무 입력이네 | S2 | "캐릭터=업무" 전환 순간 재미 |
| **오프닝** | 크레디트 재생 | 🤩 분위기 좋은데 | S3 | 한 번은 끝까지 보게 만듦 |
| **메인 사용** | 퀘스트 카드 진행률 업데이트 | 🙂 생각보다 편함 | S4 | 클리어 이펙트로 소소한 보상 |
| **재방문 (둘째 날)** | 다시 열기 → 어제 데이터 | 😎 익숙함 | S1 → S2 | 오프닝 자동 Skip |
| **장기 사용** | 일주일 뒤 | 🤔 계속 쓸까? | 전체 | **이탈 위험 구간** — 보상/루틴화 필요 |

### 용사 여정의 핵심 감정 포인트
- **"어? 업무 입력이었네"** — 캐릭터 생성 화면의 반전 (S2)
- **"클리어!"** — 진행률 100% 도달 순간 (S4)
- **"자동 스킵 되네"** — 재방문 편의 (S3)

### 용사 여정의 위험 구간
- **둘째 날 이탈**: 첫날 재미 있었어도 둘째 날 귀찮으면 안 옴
  - 대응: 입력 필드 최소화, localStorage로 어제 퀘스트 템플릿 제안
- **일주일 후 이탈**: 재미 요소 소진
  - 대응: 주간 리포트("이번 주 N개 클리어"), 장기 성취

## 보스 여정 — 박팀장

| 단계 | 액션 | 감정 | 터치포인트 | 개선 기회 |
|---|---|---|---|---|
| **인지** | 팀원 중 한 명이 "써봤는데 좋아요" 추천 | 🤨 또 관리 툴..? | 팀원 구두 | "팀원 감시 X" 메시지 강조 |
| **진입 (첫 방문)** | 보스모드 선택 | 😊 내 팀 보여줘 | S1 | 용사/보스 동등 카드 — "감시 툴" 느낌 배제 |
| **첫 대시보드** | 팀원 카드 그리드 본 순간 | 😍 한눈에 보이네 | S5 | 정보 밀도 vs 가독성 밸런스 |
| **막힘 발견** | 빨간 뱃지 팀원 카드 클릭 | 🤔 도와줘야겠다 | S5-1 | 막힘 메모가 충분히 상세해야 |
| **지원 액션** | 해당 팀원에게 DM 또는 직접 대화 | 😌 개입 완료 | 외부(슬랙 등) | 본 서비스 밖 행동 — 자연스럽게 유도 |
| **일일 루틴화** | 아침 출근하면 자동으로 열기 | 🙂 루틴 | 북마크 | 홈 스크린 또는 탭 핀 |
| **주간 정리** | 한 주 끝나고 패턴 확인 | 🤔 데이터 필요 | 전체 | **장기 미구현** — 주간 리포트 요구 |

### 보스 여정의 핵심 감정 포인트
- **"한눈에 보이네"** — 첫 대시보드 진입 (S5)
- **"아, 여기서 막혔구나"** — 막힘 뱃지 발견 (S5-1)
- **"내일 또 봐야지"** — 일일 루틴화

### 보스 여정의 위험 구간
- **정보 부족**: 데모 팀원 데이터만으로는 설득력 약함 (실제 서비스 전환 시)
- **알림 없음**: 막힘 발생을 실시간으로 모름 (데모 범위 밖)
- **1인 리더**: 팀원이 입력 안 하면 대시보드 빔 → 사용 중단

---

## 두 여정의 교차점

```
용사가 진행률 업데이트
         ↓
  localStorage 저장
         ↓
  보스 대시보드 반영
         ↓
  보스가 막힘 발견
         ↓
  외부 채널(슬랙/대화)로 지원
         ↓
  용사가 막힘 해제 → 다시 진행률 업데이트
```

**이 루프가 하루 안에 2~3번 돌면** 두 사용자 모두 가치 체감 → 이탈 방지.

**데모에서 보여줄 교차점**: 용사모드에서 입력 → 모드 변경 → 보스모드에서 바로 반영되는 것을 **한 세션 안에서** 시연.
