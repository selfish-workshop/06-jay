# CX (Customer Experience) — 퇴근을 향하여

이 폴더는 "누가 어떻게 사용할 것인가"의 상세를 담습니다.
두 페르소나의 니즈가 **정확히 맞물리는 구조**가 이 서비스의 핵심 가치 제안입니다.

## 문서 구성

| 파일 | 내용 |
|---|---|
| [`01-personas.md`](./01-personas.md) | 용사(김대리)·보스(박팀장) 2명 페르소나 |
| [`02-journey-map.md`](./02-journey-map.md) | 인지→진입→사용→재방문 여정 + 교차점 |
| [`03-faq.md`](./03-faq.md) | 14개 예상 질문과 답변 |
| [`04-onboarding.md`](./04-onboarding.md) | 첫 방문 2분 시나리오 + 이탈 대응 |

## 핵심 인사이트

1. **두 니즈가 맞물린다**: 직원은 "보고는 피곤" + 리더는 "묻기 미안" → 같은 해결책
2. **첫 2분이 전부**: 그 안에 "캐릭터 생성 = 업무 입력" 전환의 재미를 체감해야 함
3. **가치 체감의 정점**: 모드 전환 시 본인 데이터가 보스모드에 반영되는 순간
4. **감시 툴 느낌 회피**: 보스모드 읽기 전용 + 팀원 주도 입력 구조
